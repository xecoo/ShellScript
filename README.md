# Arquivo shellScript

Automatização de instalação da área de trabalho para o Ubuntu.

## O script contém os seguintes programas:
<ul>
  <li>Discord</li>
  <li>Google Chrome</li>
  <li>Kdenlive</li>
  <li>Slack</li>
  <li>Firefox</li>
  <li>Mysql</li>
  <li>Postman</li>
  <li>Skype</li>
  <li>Vs Code</li>
  <li>Zoom</li>
  <li>WhatsDesk</li>
</ul>

Certique-se do arquivo ter permissões de execução.

Para utilizar o Script, basta rodar o comando:

```
sudo ./workstation.sh
```
